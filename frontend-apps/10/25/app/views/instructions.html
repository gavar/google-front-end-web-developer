Instructions

  • Review the legoStore Controller
  • Create a "storeFront.html" template in the "views" directory
  • In the storeFront template, use expressions and directives to:
    • display the Store's name in a heading - ID of storeHeading
    • display the number of brick colors offered - ID of brickColors
    • display a list of the colors - ID of colorList
    • each color should have an ID with "color-" in front of it (e.g. color-Red, color-Blue, color-Green, etc.)
    • a color should have an item for each brick size
    • each brick size should have a class with "brick-" in front of it (e.g. brick-1x2, brick-2x4, brick-2x6, etc.)
    • for each size, display the:
      • size, quantity, and price
      • if the quantity is below 10, add the low-stock class to this item
      • if the quantity is below 5, display a message with the class coming-soon that tells the user more is on the way
      • create a "Buy" button
        • give it the class buyButton
        • create a buy function in the legoStore controller that will increase the brick's quantity by one
        • add a click handler to the button that will call the controller's buy function
      • create a "Return" button
        • give it the class returnButton
        • create a return function in the legoStore controller that will decrease the brick's quantity by one
        • add a click handler to the button that will call the controller's return function
